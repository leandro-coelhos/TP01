Descrição detalhada

Classe de teste para a classe Duracao.

A classe TUDuracao realiza testes unitários para o método setDuracao e getDuracao da classe Duracao, verificando se os valores válidos e inválidos são tratados corretamente.

Esta classe realiza testes unitários para a classe Duracao, verificando se os cenários de valores válidos e inválidos são corretamente tratados. A classe realiza a configuração do ambiente de teste antes da execução e a limpeza após a execução de cada teste.
