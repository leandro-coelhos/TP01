Descrição detalhada

Classe de teste para a classe Codigo.

A classe TUCodigo realiza testes unitários para os métodos setCodigo e getCodigo da classe Codigo, verificando o comportamento quando valores válidos e inválidos são fornecidos.

Esta classe realiza os testes unitários para a classe Codigo. O objetivo é garantir que a classe Codigo funcione corretamente para valores válidos e inválidos. O teste inclui a configuração de um objeto Codigo com valores válidos e inválidos e a verificação do comportamento esperado para cada cenário.
