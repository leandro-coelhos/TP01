Descrição detalhada

Classe de teste para a classe Destino.

A classe TUDestino realiza testes unitários para os métodos de configuração de datas da classe Destino, verificando se os valores de DataInicio e DataFim são corretamente configurados e recuperados.

Esta classe realiza testes unitários para a classe Destino. O objetivo é garantir que a classe Destino funcione corretamente quando associada a valores válidos para código, nome, datas de início e fim e avaliação. O teste inclui a configuração de um Destino com esses valores e a verificação do comportamento esperado.
