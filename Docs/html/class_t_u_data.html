Descrição detalhada

Classe de teste para a classe Data.

A classe TUData realiza testes unitários para os métodos setData e getData da classe Data, verificando se os valores válidos e inválidos são tratados corretamente.

Esta classe realiza testes unitários para a classe Data, verificando se os cenários de valores válidos e inválidos são corretamente tratados. A classe configura o ambiente de teste antes da execução e realiza a limpeza após a execução de cada teste.
