Descrição detalhada

Classe de teste para a classe Conta.

A classe TUConta realiza testes unitários para a classe Conta, verificando o comportamento do armazenamento e recuperação da senha e do código da conta por meio dos métodos setSenha, getSenha, setCodigo e getCodigo.

Esta classe realiza testes unitários para a classe Conta. O objetivo é garantir que a classe Conta funcione corretamente quando associada a uma senha e a um código válidos. O teste inclui a configuração de uma Conta com valores válidos e a verificação do comportamento esperado.
