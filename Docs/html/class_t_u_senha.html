Descrição detalhada

Classe de teste para a classe Senha.

A classe TUSenha realiza testes unitários para os métodos setSenha e getSenha da classe Senha, verificando o correto funcionamento para valores válidos e inválidos.

Esta classe realiza testes unitários para a classe Senha, verificando se os cenários de valores válidos e inválidos são corretamente tratados. A classe configura o ambiente de teste antes da execução e realiza a limpeza após a execução de cada teste.
