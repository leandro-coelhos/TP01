Descrição detalhada

Classe que representa uma Atividade.

Representa uma atividade com atributos como código, nome, avaliação, data, horário, duração e preço.

Contém informações detalhadas sobre uma atividade, como código, nome, data, horário, duração, preço e avaliação.

A classe Atividade utiliza diversos domínios para armazenar informações relacionadas à atividade. Os valores podem ser configurados e acessados por meio de métodos 'set' e 'get'.
