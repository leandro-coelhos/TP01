Descrição detalhada

Classe de teste para a classe Avaliacao.

A classe TUAvaliacao realiza testes unitários para os métodos setAvaliacao e getAvaliacao da classe Avaliacao, verificando se os valores válidos e inválidos são tratados corretamente.

Esta classe realiza testes unitários para a classe Avaliacao, verificando se os cenários de valores válidos e inválidos são corretamente tratados. A classe configura o ambiente de teste antes da execução e a limpeza após a execução de cada teste.
